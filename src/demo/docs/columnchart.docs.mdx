<a name="new_ColumnChart_new"></a>

### new ColumnChart(config)

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| config | <code>Object</code> |  |  |
| config.id | <code>string</code> |  | id of chart container div |
| config.chartTitle | <code>string</code> |  | chart title |
| config.xAxis | <code>Object</code> |  | legend level config |
| config.xAxis.type | <code>string</code> |  | type of x-axis [category|date] |
| config.xAxis.dataKey | <code>string</code> |  | key to map x-axis value |
| config.xAxis.title | <code>string</code> |  | axis title |
| config.xAxis.minGridDistance | <code>number</code> | <code>30</code> | Minimum distance between 2 labels on xAxis. This might miss align labels with column bars. Set 'dx' on labels to provide offset. See examples |
| config.xAxis.minCellDistance | <code>number</code> | <code>8</code> | Minimum distance between 2 Columns in pixels. Scrollbar will restrict zoom out to maintain this distance. |
| [config.xAxis.dateFormat] | <code>number</code> | <code>MMM dd</code> | Minimum distance between 2 Columns. Scrollbar will restrict zoom out to maintain this distance. |
| config.yAxis | <code>object</code> |  | y-axis configuration object |
| config.yAxis.title | <code>object</code> |  | Title of y-axis |
| config.yAxis.minGridDistance | <code>object</code> | <code>30</code> | Minimum distance between 2 labels on y-axis |
| config.yAxis.autoScale | <code>boolean</code> | <code>false</code> | Allows y-axis to auto scale on zoom to adjust Column height for best fit. |

**Example**  
```js
let chart = new ColumnChart({
      id: "doc-col-chart",
      chartTitle: "Category Column Series",
      xAxis: {
        type: "category",
        dataKey: "country",
        title: "Country",
        minGridDistance: 50,
      },
      yAxis: {
        title: "Visits",
      },
    });

    chart.addColumnSeries({
      name: "Country Visits ",
      value: "visits",
      clickable: true,
    });

    chart.bindData(data);
```

### columnChart.addColumnSeries(config)
Add column series to chart. Can be called multiple times to add multiple series to chart.

**Kind**: instance method of [<code>ColumnChart</code>](#ColumnChart)  

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| config | <code>Object</code> |  |  |
| config.name | <code>string</code> | <code>&quot;ColumnSeries&quot;</code> |  |
| config.value | <code>string</code> | <code>&quot;value&quot;</code> | data key to map y-axis value for the series |
| config.tootipOrientation | <code>string</code> |  | Series tooltip orientation |
| config.tooltipFill | <code>string</code> | <code>&quot;#FFFFFF&quot;</code> | Background fill color for tooltip |
| config.tooltipColor | <code>string</code> | <code>&quot;#6C757D&quot;</code> | Text color for tooltip |
| config.tooltipText | <code>string</code> | <code>&quot;{categoryX}&#x3D;[bold]{valueY}[/&quot;</code> | Pattern to show tooltip text. More here [amcharts docs](https://www.amcharts.com/docs/v4/concepts/formatters/formatting-strings/) |
| config.tooltipHTML | <code>string</code> | <code>null</code> | Pattern to show tooltip in HTML |
| [config.tooltipDateFormat] | <code>string</code> | <code>&quot;MMM dd&quot;</code> | Pattern to show tooltip in HTML |
| config.fillColor | <code>string</code> | <code>&quot;#3CA6D4&quot;</code> | Fill color for column |
| config.strokeColor | <code>string</code> | <code>&quot;#3CA6D4&quot;</code> | Stroke color for column |
| config.stacked | <code>string</code> | <code>false</code> | Set series as stacked column series. |
